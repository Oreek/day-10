# Haxmas Day - 10

A simple chrome dino like game made using Lua LOVE2D.
btw BGM and Art made by me :3

## Features
- Different background and art.
- Persistent highscore saved in a `.dat` file.
- The gameplay flips horizontally every added 500 score with change in theme.
- Custom BGM specially for this.
- Multiple obstaces (only art, the logic remains the same).
